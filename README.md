## 🎯 About The Portfolio

A premium, highly interactive personal portfolio built specifically to showcase **Software Quality Assurance (SQA)** expertise. This project demonstrates not just automation and manual testing skills, but also frontend development proficiency using modern tools like React, Vite, Tailwind CSS, and Framer Motion. 

The portfolio features a dark-themed glassmorphism UI, animated statistic counters, fully functional EmailJS integration, and interactive project showcases.

## ✨ Key Features

- 🎨 **Premium UI/UX:** Sleek dark mode with glassmorphism effects and neon accents.
- 🚀 **High Performance:** Built with Vite for lightning-fast HMR and optimized builds.
- 🎬 **Smooth Animations:** Integrated `framer-motion` for scroll animations, typing effects, and element transitions.
- 📧 **Live Contact Form:** Fully functional contact form powered by EmailJS.
- 📊 **Dynamic Data:** Easily updatable sections for Experience, Projects, and Certifications.
- 📱 **Fully Responsive:** Flawless rendering on mobile, tablet, and desktop devices.

## 💻 Local Installation & Setup

To run this portfolio locally on your machine, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/oynndri/MyPortfolio.git
   cd MyPortfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure EmailJS (Optional)**
   Open `src/components/Contact.jsx` and replace the placeholder credentials with your EmailJS keys if you want the contact form to work.

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```
